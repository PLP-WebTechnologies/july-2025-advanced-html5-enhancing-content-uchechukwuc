How to Brew a Perfect Cup of Coffee (Ordered List)

  1. Boil water to the correct temperature (about 90-96°C).
  2. Grind your coffee beans to a medium-fine consistency.
  3. Place a filter in your dripper and add the ground coffee.
  4. Pour hot water over the grounds to 'bloom' them for 30 seconds.
  5. Slowly pour the remaining water in a circular motion.
  6. Enjoy your freshly brewed coffee! ☕

Key Web Terms (Definition List)

HTML
HyperText Markup Language, the standard language for creating web pages.
CSS
Cascading Style Sheets, a language used to describe the presentation of a document written in HTML.
DOM
Document Object Model, a programming interface for web documents that represents the page so that programs can change the document structure, style, and content.
